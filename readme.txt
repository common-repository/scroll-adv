== Description ==

Display your advert on left and right your website.Scroll when mouse scroll.
== Installation ==

1. Upload Scroll Adv to your plugins directory, then activate the plugin through the 'Plugins' menu in WordPress.
2. You can also place Scroll Adv in your template or theme via the PHP command scroll_adv()
`<?php scroll_adv(); ?>` after  `<body>` tag in index.php or home.php file and after that go to widget management to add your advert.
